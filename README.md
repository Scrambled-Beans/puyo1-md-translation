# Puyo Puyo (Mega Drive) - Translation

This is a partial disassembly to translate Puyo Puyo for the Sega Mega Drive into English.


## Translating Puyo Puyo to Another Language

You are welcome to use the source code to translate the game to another language. If you need any help, feel free to contact RadioTails. The best way is to send an e-mail to: radiotailspuyo@outlook.com


## Translating Other Versions of Puyo Puyo

As long you credit the required people, then I have no problem with the script or edited graphics being used to translate other versions of Puyo Puyo.


## Original Puyo Puyo (Mega Drive) ROM

You will need to obtain a copy of the "Puyo Puyo (Japan)" ROM (.md or .bin).<br/> 

Use MD5-SHA-Checksum-Utility (https://github.com/oop7/MD5-SHA-Checksum-Utility/releases) to check the "MD5" and "SHA-1" match the following:<br/> 
- MD5: c723d1079157ab78e160b26ac5091d74<br/> 
- SHA-1: 02656b5707cf9452d4cf48378ffd3a95dc84e9c5


## Building (Windows)

1 - Download the repo.<br/>
2 - Place the original Puyo Puyo (Mega Drive) ROM in the "rom" folder.<br/> 
3 - Rename the file to: Puyo Puyo.bin<br/> 
4 - Open "Puyo Puyo (Translation).asm" and adjust any settings.<br/>
5 - Run "build.bat" to build the rom.<br/>
6 - The built rom can be found in the "output" folder.
